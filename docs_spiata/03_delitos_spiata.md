# Tipificación de Delitos (Ley N° 21.459)

**Art. 1°: Acceso Ilícito**
- Ingreso no autorizado a sistemas de BancoEstado
- Vulneración de barreras de autenticación

**Art. 3°: Sabotaje Informático**
- Cifrado masivo de datos con Sodinokibi
- Bloqueo total del funcionamiento de sistemas
- Impidió operación de 410 sucursales

**Art. 5°: Receptoración de Datos**
- Extracción de datos confidenciales
- Modalidad: doble extorsión
- Venta/retención ilícita de información
