# Datos Personales y Derechos ARCO

**Datos Comprometidos:**
- Personales: Nombres, RUT, correos, domicilios, teléfonos
- Sensibles: Historial de transacciones, saldos, deudas

**Derechos ARCO Afectados:**
- **Acceso:** Imposible - sistemas cifrados por ransomware
- **Cancelación:** Vulnerada - datos retenidos por atacantes
- **Oposición:** Violada - datos expuestos a terceros no autorizados

**Conclusión:** Violación grave de privacidad de 13M de clientes
